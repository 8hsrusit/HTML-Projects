----------Video----------


29 - <video><video> / ----- to add video(s) to your Website

                    NOTE : Its like the img tag For Example

                          <video>
                           <source src="vid source"
                           type="vid_extantion">   
                          </video>

                    NOTE -2- : you must use type attribute to define the video extantion
                    such as ( MP4 , webm , MKV , etc.... )

                    IMPORTANT-NOTE : different media formats are needed for compatibility
                    with different browsers and devices. the <source> tag allows you 
                    to add multiple files. for Example

                     <video>
                           <source src="vid source.mp4"
                           type="video/mp4">
                             <source src="vid source.webm"
                           type="video/webm">

                     </video>
 

                   NOTE -3- : you can add text if the video wont load on browser the text
                   will appear for Example

                     <video>
                           <source src="vid source.mp4"
                           type="video/mp4">
                             <source src="vid source.webm"
                           type="video/webm">
                           Video not supported

                     </video>
 
                  NOTE -4- : You can add "video controls" ( Play/pause , Volume , etc ..)
                  for Example
         
                     <video controls>
                           <source src="vid source.mp4"
                           type="video/mp4">
                             <source src="vid source.webm"
                           type="video/webm">

                     </video>



                
-----------Audio-----------


30 - <audio></audio> / ------ To add any audio to your website

                     NOTE : you can use it like video TAG for Example
                        
                     <audio>
                           <source src="audio source.mp3"
                           type="audio/mpeg">
                             
                     </audio>

                       
                   NOTE -2- : "mpeg" belong to mp3 format

                   NOTE -3- : you can add multiple audio like the video TAG because
                   it use the source TAG

                   NOTE -4- : like video you must use type attribute to define the 
                   audio Extension suchb as ( mp3 , ogg , wav , etc .... )

                   NOTE -5- :  you can add text if the audio wont load on browser the text
                   will appear for Example

                     <audio>
                           <source src="au source.mp3"
                           type="audio/mpeg">
                             <source src="au source.ogg"
                           type="audio/ogg">
                           Audio type not supported
                     </audio>

                   
                  NOTE -6- : you can add "audio controls" ( Play/pause , Volume , etc ..)
                  for Example
                
                      <audio controls>
                           <source src="au source.mp3"
                           type="audio/mpeg">
                             <source src="au source.ogg"
                           type="audio/ogg">
                           Audio type not supported
                     </audio>

 
                NOTE -7- : you can use attribute like ( loop , autoplay , muted )

                NOTE -8- : "loop" to replay the audio when the audio ends
 
                           "autoplay"to play the audio when the page load

                           "muted" to play the audio whrn the page load but
                           play in mute

               Example :

                    <audio autoplay>
                           <source src="au source.mp3"
                           type="audio/mpeg">
                             <source src="au source.ogg"
                           type="audio/ogg">
                           Audio type not supported
                    </audio>

                NOTE -9- : you can use more than one attribute for Exmple


                    <audio autoplay muted>
                           <source src="au source.mp3"
                           type="audio/mpeg">
                             <source src="au source.ogg"
                           type="audio/ogg">
                           Audio type not supported
                    </audio>
             
                Explain : the audio will play when the page loads but with no sound


                NOTE -10- : you can use these attributes with audio && video

40 - colspan , Rowspan attribute / ------ you can make cells that take up 
     multiple rows and/or columns, using the attributes colspan and/or
     columns, using the attributes colspan and rowspan. this is called
     merging cells

     A - colspan / ---- colspan is for the number of columns a cell should 
         span
     
     NOTE : you could understand it like this its extend like this "-------"
     i mean horizontal

     Example :  

                           <table>
                           <th>Name</th><th>Age</th>
                            <tr>
                              <td>Joseph</td><td>16</th>
                            </tr>
                            <tr>
                              <td>Mahmood</td><td>17</td>
                            <tr>
                             <td colspan="2">Rana-18</td>
                           </table>

     B - rowspan / ----- the rowspan attribute specifies the number of rows
         a cell should span
          
         NOTE -2- : you can uderstand it like this "|
                                                    |
                                                    |
                                                     " i mean vertically
                                                       for Example             


                       <table>
                         <th>Year</th>
                         <th>Month</th>
                         <th>Day</th> 
                          <tr>
                            <td>2026</td>
                            <td>June</td>
                            <td rowspan="2">23</td>
                          </tr>
                       </table>
