--------Priorities-------

EXPLAIN : CSS Has Some Rules and Priorities for appling properties the Priorities are

            1A - the most ==> Inline elements 
            2B -  2nd most ==> IDs
            3C - 3rd most ==> class
            4D - the least ==> element

        NOTE : this Priorities will apply if the same element Has multiple properties 
                For Example
              
            <style>
            p {
              color:red;
            }

            #int {
              color:green;
            }

            .intro {
              color:blue;
            }
            </style>

              <body>
               <p id="int" class="intro">YAP</p>
              </body>

        EXPLAIN : the <p> will apply green color Because the Priorities for IDs is the Most

        Q : What is the Inline Elements?
        
        A : the element that has his own attribute and it have the Higher Priorities on CSS
            For Example

               <p style="color:violet">YAP2</p>

        EXPLAIN -2- : this TAG has its own attribute "style" so CSS will ignore Any IDs,class and
                      Element properties in <Style> and will apply the element attribute "style"
                      Because it have the higher Priorities

11 - text-Decoration ==> to add Decoration for your text soe of these Decoration are :

                            A1 => underline ==> to add line under your text
                            B2 => overline ==> to add a line upper your text
                            C3 => line-through => fo add a line through your Text
                            D4 => none ==> to remove any Decoration like <a href="url">yap</a>

                Examples :

                 <p id="of">
                 Limited Time Offer!
                 <b> Try our New </b>
                 </p>

                 <!-- I want to target <b> --> 
                 
                 /*CSS*/

                 #of b {
                  text-decoration:underline;
                 }          

          EXPLAIN : we're telling CSS to go to the element with id of "of" which is <p> -parent-
                    and apply these properties in <b> -child- 

          NOTE : some CSS properties can hav multiple values you can control the color for the
                 decoration by adding a color , RGB or HEX for Example 

                 #of b {
                  text-decoration:overline #50003c;
                 }

          NOTE -2- : you can use styles on Decoration such as 
                     
                     A1 => dotted ==> make your decoration dotted
                     B1 => double ==> make your decoration doubled
                     C3 => dashed ==> make your decoration dashed
                     D4 => wavy ==> make your decoration wavy

          Examples :

             #of b {
              text-decoration:line-through #479260 wavy
             }

12 - text-transform ==> allows you to control the capitalization of the text content such as

                  Examples :
